# Requirements and Installation
#### Requirements
* **[Python 3.6 or higher](https://www.python.org/downloads/)**
#### Installation and Use
1. Install requirements
```bash
    $ pip install -r requirements.txt
```
2. Just start!
```bash
    $ python CRIScrape.py
```
